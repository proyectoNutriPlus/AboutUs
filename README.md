# NutriPlus

## Objetivo del proyecto

Realizar un SaaS que permitirá a los nutricionistas registrar y realizar seguimiento del estado nutricional de sus pacientes, calcular automáticamente (IMC, porcentaje de peso graso, requerimientos calóricos), generar planes de alimentación personalizados según patologías específicas, y realizar un monitoreo continuo de la evolución del tratamiento nutricional.
